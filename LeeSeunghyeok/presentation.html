데이터 수집 및 정제

1. 데이터 수집

  • 출처: 각 대학 입학처의 공식 논술 가이드북
  • 내용: 기출문제, 출제의도, 채점기준, 모범답안
  • 주의: 교육 및 연구 목적으로만 사용 (상업적 이용 불가)

2. 데이터 정제

  • 문제 추출: 사용자가 문제를 확인할 수 있도록 PDF로 별도 관리
  • 핵심 정보 구조화: 출제의도, 채점기준, 모범답안을 JSON 파일로 생성
  • 최종 변환: JSON 데이터를 LangChain Document 객체로 변환하여 활용
